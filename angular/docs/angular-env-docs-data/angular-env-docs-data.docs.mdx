description: Constants for populating the Angular section of bit's documentation.
---

A component that provides data such as the Angular env Name, templates etc, required for hydrating bit's Angular documentation

### Component usage
```js
import { angularEnvDocsData } from '@bitdev/angular.docs.angular-env-docs-data';

<SomeDocsSectionMdxComponent {...angularEnvDocsData} />

```
